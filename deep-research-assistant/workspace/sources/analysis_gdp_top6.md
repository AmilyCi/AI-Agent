## 四、关键结论

1. **六省贡献近半**：GDP前6省份合计55.78万亿元，占全国44.25%，以约19.4%的省级行政区数量贡献了接近一半的全国经济总量。

3. **增速分化显著**：
   - 山东、浙江、四川三省并列增速第一（6.0%），均高于全国均值0.8个百分点

   - 六省平均增速5.45%，高于全国均值0.25个百分点

4. **第5名之争**：四川（60,132.90亿元）以仅1,000.51亿元的微弱优势超过河南（59,132.39亿元），且四川增速6.0%远高于河南4.1%，差距有扩大趋势。

5. **头部差距悬殊**：广东GDP是河南的2.29倍，首尾差距达7.65万亿元。

## 五、计算代码（可复现）

```javascript
const provinces = [
  { name: '广东', gdp: 135673.16, growth: 4.8 },
  { name: '江苏', gdp: 128222.20, growth: 5.8 },
  { name: '山东', gdp: 92069.00,  growth: 6.0 },
  { name: '浙江', gdp: 82553.00,  growth: 6.0 },
  { name: '四川', gdp: 60132.90,  growth: 6.0 },
  { name: '河南', gdp: 59132.39,  growth: 4.1 },
];
const nationalGDP = 1260582;
const nationalGrowth = 5.2;

// 六省总和
const totalGDP = provinces.reduce((sum, p) => sum + p.gdp, 0);
// → 557782.65 亿元

// 各省占比
provinces.forEach(p => {
  p.share = (p.gdp / nationalGDP) * 100;
});
// 六省合计占比 → 44.2480%

// 增速排名
const sorted = [...provinces].sort((a, b) => b.growth - a.growth);
// 山东=浙江=四川(6.0%) > 江苏(5.8%) > 广东(4.8%) > 河南(4.1%)

// 六省平均增速
const avgGrowth = provinces.reduce((sum, p) => sum + p.growth, 0) / 6;
// → 5.45%
```
